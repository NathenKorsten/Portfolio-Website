About Me

Projects

Vanilla JS

Weather App

Fully responsive and purely javascript, this weather app uses the OpenWeather API to display the weather of a user specified city. In addition to this, the API provides icons relating to certain types of weather, which is also displayed on the app. Together with the temperature, the app also discloses the dew point and sea level.

Random User Generator

Back-end developers often need dummy data to test routes. This fully responsive app uses the Random.me API to generate random user data, including the most common route testing requirements, such as a password, email, location, avatar, name and surname, and login. It is written in vanilla JS and is presented with a simple and easy to read UI.

URL to QR code Converter

This app takes in a URLs and converts it into a downloadable QR code. It uses the image-charts API to generate QR codes from URLs. In addition to URLs, it also converts text to QR codes.

Covid-19 Dashboard

A simple yet informative app that presents South-Africa's most important COVID-19 statistics, which includes deaths, recoveries and infections. It uses the COVID-19 API, and is always provides the latest statistics

Map App

Using the MapBox API, this app resembles a google map variant, allowing for obtaining current locations, navigation, and directions. In addition, it is very lightweight and purely vanilla JS

Codepen Clone

Codepen is one of the most used coding playgrounds available. This clone offers most of the general codepen features, such as automatic saving and rendering, 3 tabs for HTML, CSS, and javascript respectively. Amazon Clone

Spotify Clone

Using Spotify's API, REACT Context API and reusable components, this app allows for authentication and pulls respective user data such as playlists and what the user is currently listening to. Playing music and general audio functionality is currently in progress

Google Drive Clone

This clone uses firebase for storage, and database collections in effort to produce a google drive clone. Equipped with logging in, authentication and registration, this REACT clone has most of Google Drive's functionality, including file storage, folder structure, password resetting and much more.
